## 3) 라운드 1 미해결 보완 항목 고정 리스트

### QA 3건

1. 요구사항-테스트 추적매트릭스(RTM) v1 작성
- 오너: 호크(작성·취합), 기능별 QA 담당(검증)
- 마감: 2026-02-20 17:00 KST
- 완료기준: 요구사항 ID 대비 테스트케이스 100% 매핑, 누락 0건
- 증빙: RTM 링크 + 누락 0건 체크리스트

2. CI 스모크 자동화 기준 합의
- 오너: 호크 + 개발팀 아리아
- 마감: 2026-02-21 14:00 KST
- 완료기준: 스모크 기준 정의 문서 승인 + 첫 파이프라인 PASS
- 증빙: 기준 합의 문서 링크 + PASS 로그

3. 결함 심각도/SLA/종료조건 확정
- 오너: 호크
- 마감: 2026-02-21 16:00 KST
- 완료기준: Blocker/Critical 0건 종료조건 합의본 승인
- 증빙: Severity/SLA/Exit Criteria 합의본 링크

- 마감: 2026-02-21 14:00 KST
- 완료기준: 단계별 담당자, RTO/RPO 명시
- 증빙: 런북 v1 링크

2. 모니터링·알림 기준선 합의

- 마감: 2026-02-21 16:00 KST

3. 온콜·장애 에스컬레이션 체계 등록
